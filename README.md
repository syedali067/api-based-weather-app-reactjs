# 🌤️ Weather App (React + OpenWeather API)

A modern weather application built with **React.js** that provides real-time weather data, 5-day forecasts, geolocation-based weather detection, and dark mode support.

## ⚙️ Features

* 🔍 Search weather by city name
* 📍 Auto-detect user location (GPS)
* 🌡️ Real-time weather data (OpenWeather API)
* 📅 5-day weather forecast
* 🌙 Dark / Light mode toggle
* ⚡ Fast and responsive UI
* 📱 Mobile-friendly design
* ❌ Error handling (invalid city / location denied)

---

## 🛠️ Tech Stack

* React.js (Vite)
* JavaScript (ES6+)
* CSS3
* OpenWeather API

## 📦 Installation & Setup

### 1. Clone the repository

```bash
git clone https://github.com/your-username/api-based-weather-app-reactjs.git
```

### 2. Move into project folder

```bash
cd api-based-weather-app-reactjs
```

### 3. Install dependencies

```bash
npm install
```

### 4. Run the project

```bash
npm run dev
```

---

## 🔑 API Setup

This project uses OpenWeather API.

1. Get your API key from:
   👉 [https://openweathermap.org/api](https://openweathermap.org/api)

2. Replace API key in `App.jsx`:

```js
const API_KEY = "your_api_key_here";
```

## 🌍 Features Explained

### 📍 Geolocation Weather

Automatically detects user location using browser GPS and shows current weather.

### 📅 5-Day Forecast

Displays weather forecast using OpenWeather 5-day API (filtered per day).

### 🌙 Dark Mode

Toggle between light and dark UI for better user experience.
